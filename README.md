# TUX the Penguin

This project is a small exercise centered around **Tux**, the well-known mascot of the Linux operating system.  
It includes different files demonstrating basic web content, documentation formats, and a simple shell script.

## Project Structure

- **tux.html**  
  A simple HTML page related to Tux.

- **tux.pdf**  
  A PDF document containing information or material about Tux.

- **tux.rst**  
  A documentation file written in **reStructuredText**, commonly used in technical documentation.

- **mohamedyoussef.sh**  
  A shell script demonstrating basic scripting in a Unix/Linux environment.

## Purpose

The goal of this project is to practice working with:

- Basic web content (HTML)
- Documentation formats
- Shell scripting
- Version control with Git and GitHub

## Technologies Used

- HTML
- Shell scripting (Bash)
- reStructuredText
- Git & GitHub
